# JavaScript Calculator

A modern, responsive calculator built with vanilla JavaScript, HTML, and CSS.

## Features

- **Basic Arithmetic Operations**: Addition (+), Subtraction (-), Multiplication (*), Division (/)
- **Decimal Support**: Handle decimal numbers with precision
- **Clear Function**: Clear all input with the 'C' button
- **Backspace**: Remove the last entered digit with the '⌫' button
- **Keyboard Support**: Use your keyboard for input
- **Modern Design**: Glassmorphic design with smooth animations
- **Responsive**: Works on desktop and mobile devices

## How to Use

### Mouse/Touch Input
1. Click the number buttons (0-9) to enter numbers
2. Click the operator buttons (+, -, *, /) to perform operations
3. Click the equals button (=) to calculate the result
4. Click 'C' to clear the display
5. Click '⌫' to delete the last character

### Keyboard Input
- **Numbers**: Type 0-9 to enter numbers
- **Operators**: Type +, -, *, / for operations
- **Calculate**: Press Enter or = to get the result
- **Clear**: Press Escape or Delete to clear
- **Backspace**: Press Backspace to delete the last character
- **Decimal**: Press . for decimal point

## Getting Started

1. Open `index.html` in your web browser
2. Start calculating!

## Files

- `index.html` - Main HTML structure
- `calculator.js` - JavaScript functionality
- `style.css` - CSS styling and animations

## Browser Compatibility

Works in all modern browsers including Chrome, Firefox, Safari, and Edge.
